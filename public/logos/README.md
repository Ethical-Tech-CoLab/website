# Client & partner logos

Drop a logo file here and it appears automatically on the About page — no code
changes needed. The site looks for `/logos/<slug>.<ext>` where `<slug>` is the
org name lowercased with non-letters turned into dashes. If no file is found it
falls back to a lettered monogram.

Supported extensions (checked in this order): `.svg`, `.png`, `.webp`, `.jpg`.
Square images (or with transparent padding) look best; they render on a white
rounded tile.

## Expected filenames

Clients:
- `generative-ai-for-good.png` — Generative AI for Good
- `d-id.png` — D_ID
- `mesur-io.png` — Mesur.io
- `supplytrace.png` — SupplyTrace
- `rivr.png` — Rivr
- `human-generated-by-blockapps.png` — Human Generated by BlockApps

Founding partners:
- `microsoft.svg` — Microsoft (already added)
- NYU SPS + CGA use `/nyu-sps-cga-logo.jpg` (already added)

Partners:
- `100x.png` — 100x
- `apne-aap-women-worldwide.png` — Apne Aap Women Worldwide
- `gaia.png` — Gaia
- `grid-bank.png` — Grid Bank

To use a different name for any org, set its `logo` field in
`src/content/site.ts` to the exact path (e.g. `logo: "/logos/blockapps.png"`).
